Users Guide

After administrator login, he can select any link from Menu:

1) Click on User Mgt Menu link for User management.
· Click on Create New User to create a new user by quick Registration.
· If you are a Admin type user then you have rights to access all the features, where in User type you would be allowed to use Data Collection part only.
· Click on Delete User to delete user.
· Click on Reset Password and Get Password button to reset and retrive password for the selected user.

2) Click on Prepare Form Menu link to create form.
· Click on Create New Form menu link.
· Enter Form Name, Second language name or "none" and click on "Create Form" button.
· Click on Select Form link to select Form from combo
· Click on Add Question to form button, will open question layout.
· Enter your question and select your choice accordingly.
· Click on "Add More Question" button will add your question into the database and open new question layout.
· Click on "Quit with Saving" will save your question as well as Form and you will be redirected to Prepare Form page.
· Click on "Quit without Saving" will save only your Form and you will be redirected to Prepare Form page.
·When ever your quit from add question you will be redirected to the page where you can Remove Question,Modify Question and Add Question for the currently selected Form.
· Click on Modify Language to modify lanagueg for the selected Form.
· Click on Remove Form to remove Form from Database.


3) Click on Modify Form Menu link to modify any question.
· Select Form from combo and Click on Next button to select question tag.
· Click on Remove Form button to remove entire form from Database.
· Select question tag from combo and now click on View Question button to modify.
· Modify your question.
· Click on Update Question button to save your changes. You will be redirected to select another question web Page.
· Repeat the same step to update another question.
· Click on Remove Question button to remove question for the selected Form.

4) Click on Data Collection Menu link to collect data for selected Form.
· Select Form from combo and to view Form just click on "Next>>" button.
· Select your language and enter all other Initial screen information and click on "Start Data Collection".
· Confirm answer to move to the next question.
· After last question you would get a screen saying. "This
evaluation is over. Thanks for your time".

5) Click on Query on Form Menu link to search on Data.
· Select Form from combo and just click on "Next>>" button.
· Select question from combo and click on "View Result" button.
· View result.
· Press on " <<BACK" to select another question.
· Click on Delete All Data to remove all data collected so far for the selected Form.

6) Click on Logout link ,will redirect you to login page.

7) Click on Help link , will open help file.
